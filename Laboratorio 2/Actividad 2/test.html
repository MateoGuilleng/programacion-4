🧪 Prueba de la función handleNestedObject

Nota: Este archivo HTML permite probar la función handleNestedObject en el navegador. La función implementa inmutabilidad para objetos anidados.

Para ejecutar las pruebas, haz clic en el botón "Ejecutar Pruebas" a continuación.

📋 Código de la función

La función handleNestedObject está implementada en handleNestedObject.js. Características principales:

  • ✅ Inmutabilidad garantizada
  • ✅ Manejo de objetos anidados en múltiples niveles
  • ✅ Operaciones: actualizar, agregar, eliminar
  • ✅ Copias profundas automáticas
  • ✅ Compatible con arrays anidados

Ejemplo de uso:

const updatedProfile = handleNestedObject(userProfile, {
  address: { city: "New City" },
  social: { facebook: "updatedFacebook" }
});

🧪 Panel de Pruebas

  2. Se crea un nuevo objeto con los cambios
  3. Las propiedades anidadas se actualizan correctamente
