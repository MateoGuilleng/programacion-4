📝 Ejemplo del Laboratorio: handleNestedObject

Nota: Este ejemplo implementa exactamente el caso del laboratorio, pero con una modificación:

La propiedad hobbies se cambió de array ["Reading", "Hiking"] a objeto {primary: "Reading", secondary: "Hiking"} porque nuestra función handleNestedObject solo admite objetos, no arrays.

⚠️ Importante: Nuestra función handleNestedObject es estricta y SOLO admite objetos anidados. Rechaza arrays en cualquier nivel.

📋 Código JavaScript

Este es el código del ejemplo del laboratorio, adaptado para nuestra función:

// Objeto original del laboratorio (modificado: hobbies como objeto, no array)
let userProfile = {
  id: 1,
  name: "John Doe",
  address: {
    street: "123 Main St",
    city: "Anytown",
    zip: "12345"
  },
  hobbies: {  // Cambiado de array a objeto
    primary: "Reading",
    secondary: "Hiking"
  },
  social: {
    twitter: "@johndoe",
    facebook: "johndoe123"
  }
};

// Updating a nested property and adding a new property
const updatedProfile = handleNestedObject(userProfile, {
  address: { city: "New City" },
  social: { facebook: "updatedFacebook" }
});

console.log("Objeto original:", userProfile);
console.log("Objeto actualizado:", updatedProfile);
console.log("¿Es el mismo objeto?", userProfile === updatedProfile);
console.log("¿Es el mismo objeto address?", userProfile.address === updatedProfile.address);
console.log("¿Es el mismo objeto social?", userProfile.social === updatedProfile.social);
