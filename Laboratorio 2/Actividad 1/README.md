# Custom Array Methods

Custom implementations of JavaScript array methods (map, reduce, filter, forEach) demonstrating functional programming principles.

## Project Structure

```
src/           - Source code directory
├── index.js   - Public API entry point
tests/         - Test files
examples/      - Usage examples
```

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run tests:
```bash
npm test
```

## Features

- **customMap**: Custom implementation of Array.prototype.map
- **customReduce**: Custom implementation of Array.prototype.reduce  
- **customFilter**: Custom implementation of Array.prototype.filter
- **customForEach**: Custom implementation of Array.prototype.forEach

## Development

This project follows functional programming principles:
- Pure functions where possible
- Immutability (no side effects on input arrays)
- Higher-order functions
- Comprehensive property-based testing
